# A2Z Futures Pit

A simple single-player browser game based on the A2Z Corp futures pit-market classroom exercise.

You are the only human player. The other traders are bots with Trader, Speculator, and Market Maker roles. The app runs fully in the browser and can be hosted directly with GitHub Pages.

## Play locally

Open `index.html` in a browser.

## Host on GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Set the source to the repository branch and root folder.
4. Open the published Pages URL.

No build step or server is required.
